### Fayl tizimi

**Root** `/` - barcha fayllar bu yerdan boshlanadi.

```
/
├── bin        # Asosiy buyruqlar (ls, cat, cp)
├── boot       # Boot loader fayllari
├── dev        # Device fayllari
├── etc        # Konfiguratsiya fayllari
├── home       # Foydalanuvchilar papkalari
│   ├── user1
│   └── user2
├── lib        # System libraries
├── media      # Съёмные устройства
├── mnt        # O'rnatish uchun
├── opt        # Qo'shimcha dasturlar
├── proc       # Jarayon ma'lumotlari
├── root       # Root foydalanuvchi home
├── sbin       # System buyruqlar
├── tmp        # Vaqtinchalik fayllar
├── usr        # Foydalanuvchi dasturlari
└── var        # O'zgaruvchan fayllar (logs)
```

**Muhim papkalar**:
- `/home/username` - foydalanuvchi fayllari
- `/etc` - konfiguratsiya
- `/var/log` - log fayllar
- `/tmp` - vaqtinchalik
- `/usr/bin` - dasturlar

### Terminal asoslari

#### Navigatsiya

**pwd** - hozirgi papka (Print Working Directory):
```bash
pwd
# /home/user
```

**ls** - fayllarni ko'rish (List):
```bash
ls                 # Oddiy
ls -l              # Batafsil (long format)
ls -la             # Yashirin fayllar bilan (all)
ls -lh             # Human-readable size
ls -ltr            # Vaqt bo'yicha (reverse)
```

**ls -l** output:
```
-rw-r--r-- 1 user group 1234 Jan 15 10:30 file.txt
│││││││││  │  │    │    │    │         │       └─ File nomi
│││││││││  │  │    │    │    │         └───────── Vaqt
│││││││││  │  │    │    │    └─────────────────── O'lcham
│││││││││  │  │    │    └──────────────────────── Guruh
│││││││││  │  │    └───────────────────────────── Ega
│││││││││  │  └────────────────────────────────── Linklar soni
│││││││││  └───────────────────────────────────── Fayl turi
││││││└└────────────────────────────────────────── Ruxsatlar
```

**cd** - papka o'zgartirish (Change Directory):
```bash
cd /home/user      # Aniq yo'l
cd documents       # Nisbiy yo'l
cd ..              # Yuqoriga
cd ../..           # 2 ta yuqoriga
cd ~               # Home papka
cd -               # Oldingi papka
```

#### Fayl operatsiyalari

**touch** - bo'sh fayl yaratish:
```bash
touch file.txt
touch file1.txt file2.txt file3.txt
```

**mkdir** - papka yaratish (Make Directory):
```bash
mkdir newfolder
mkdir -p dir1/dir2/dir3   # Ota-papkalar bilan
mkdir folder1 folder2 folder3
```

**cp** - nusxalash (Copy):
```bash
cp file.txt copy.txt           # Fayl
cp -r folder1 folder2          # Papka (recursive)
cp file.txt /home/user/        # Boshqa joyga
cp *.txt backup/               # Ko'p fayllar
```

**mv** - ko'chirish yoki nom o'zgartirish (Move):
```bash
mv oldname.txt newname.txt     # Nom o'zgartirish
mv file.txt /home/user/        # Ko'chirish
mv *.txt documents/            # Ko'p fayllar
```

**rm** - o'chirish (Remove):
```bash
rm file.txt                    # Fayl
rm -r folder                   # Papka (recursive)
rm -rf folder                  # Majburiy (force)
rm *.txt                       # Pattern
```

**Ehtiyot bo'ling!** `rm -rf /` - butun tizimni o'chiradi!

#### Fayl ko'rish

**cat** - fayl ko'rish (Concatenate):
```bash
cat file.txt
```

#### Qidirish

**find** - fayl qidirish:
```bash
find /home -name "*.txt"       # Nom bo'yicha
find . -type f                 # Faqat fayllar
find . -type d                 # Faqat papkalar
find . -size +1M               # 1MB dan katta
find . -mtime -7               # 7 kun ichida o'zgargan
```

**grep** - matn ichida qidirish:
```bash
grep "search" file.txt         # Oddiy qidirish
grep -i "search" file.txt      # Case-insensitive
grep -r "search" /path         # Recursive
grep -n "search" file.txt      # Qator raqami bilan
grep -v "search" file.txt      # Teskari (exclude)
grep -c "search" file.txt      # Soni
```

### Ruxsatlar (Permissions)

**3 ta tur**:
- **r** (read) - o'qish (4)
- **w** (write) - yozish (2)
- **x** (execute) - bajarish (1)

**3 ta guruh**:
- **u** (user) - ega
- **g** (group) - guruh
- **o** (others) - boshqalar

**Misol**:
```
-rw-r--r--
│││ │ │ │
│││ │ │ └── Others: r-- (read only)
│││ │ └──── Group: r-- (read only)
│││ └────── User: rw- (read, write)
││└──────── Guruh birikma
│└───────── Fayl turi (- = file, d = directory)
└────────── Special bits
```

**chmod** - ruxsat o'zgartirish:

**Symbolic**:
```bash
chmod u+x file.sh              # User ga execute
chmod g-w file.txt             # Group dan write olib tashlash
chmod o+r file.txt             # Others ga read
chmod a+x file.sh              # All ga execute
chmod u+rwx,g+rx,o+r file.txt  # Kombinatsiya
```

**Numeric** (Octal):
```bash
chmod 755 file.sh              # rwxr-xr-x
chmod 644 file.txt             # rw-r--r--
chmod 777 file.txt             # rwxrwxrwx (hammaga hamma narsa)
chmod 600 private.txt          # rw------- (faqat ega)
```

**Hisoblash**:
- r = 4
- w = 2
- x = 1

**Misol**: `755` = `rwxr-xr-x`
- 7 = 4+2+1 = rwx (user)
- 5 = 4+0+1 = r-x (group)
- 5 = 4+0+1 = r-x (others)

**chown** - ega o'zgartirish:
```bash
chown user file.txt            # Ega
chown user:group file.txt      # Ega va guruh
chown -R user folder           # Recursive
```

### Pipe va Redirection

**Pipe `|`** - output ni input ga:
```bash
ls -l | grep ".txt"
cat file.txt | wc -l
ps aux | grep firefox
```

**Redirection**:

**`>`** - yozish (overwrite):
```bash
ls > file.txt
echo "Hello" > file.txt
```

**`>>`** - qo'shish (append):
```bash
ls >> file.txt
echo "World" >> file.txt
```

**`<`** - input sifatida:
```bash
wc -l < file.txt
```

**`2>`** - error ga:
```bash
command 2> error.log
```

**`&>`** - hammasi (output va error):
```bash
command &> all.log
```
