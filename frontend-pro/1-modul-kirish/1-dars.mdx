## Dars mazmuni

### Frontend Development nima?

**Frontend Development** - veb-saytning foydalanuvchi ko'radigan va o'zaro aloqada bo'ladigan qismini yaratish. Bu saytning ko'rinishi, dizayni va interaktivligidir.

### Web Development turlari

#### 1. Frontend Development
- **Vazifasi**: Foydalanuvchi interfeysi (UI)
- **Texnologiyalar**: HTML, CSS, JavaScript
- **Ko'rish mumkin**: Ha, brauzerdagi hammasi
- **Misol**: Tugmalar, menyular, animatsiyalar, formalar

#### 2. Backend Development
- **Vazifasi**: Server, ma'lumotlar bazasi
- **Texnologiyalar**: Python, PHP, Node.js, Java
- **Ko'rish mumkin**: Yo'q, server tomonida ishlaydi
- **Misol**: Foydalanuvchilarni saqlash, parollarni tekshirish

#### 3. Full Stack Development
- Frontend + Backend
- Hamma narsani biladi va qiladi

### Frontend texnologiyalari

#### 1. HTML (HyperText Markup Language - Gipermatn belgilash tili)
- **Vazifasi**: Sahifaning tuzilmasi (struktura)
- **Misol**: Sarlavha, paragraf, rasm, havola
- **Fayl kengaytmasi**: `.html`

```html
<h1>Salom Dunyo!</h1>
<p>Bu mening birinchi veb-sahifam</p>
```

#### 2. CSS (Cascading Style Sheets - Kaskadli uslublar jadvali)
- **Vazifasi**: Sahifaning ko'rinishi (dizayn)
- **Misol**: Rang, o'lcham, joylashuv, shrift
- **Fayl kengaytmasi**: `.css`

```css
h1 {
  color: blue;
  font-size: 32px;
}
```

#### 3. JavaScript (JS)
- **Vazifasi**: Sahifaning harakati (interaktivlik)
- **Misol**: Tugma bosilganda, animatsiya, dinamik kontent
- **Fayl kengaytmasi**: `.js`

```javascript
document.querySelector('button').onclick = function() {
  alert('Tugma bosildi!');
}
```
