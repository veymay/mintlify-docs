## Dars mazmuni

### JavaScript nima?

**JavaScript** - web sahifalarni interaktiv qiluvchi dasturlash tili.

**Vazifalar**:
- Sahifani dinamik o'zgartirish
- Foydalanuvchi bilan o'zaro aloqa
- Ma'lumotlarni qayta ishlash
- Server bilan aloqa (AJAX)

**JavaScript qo'shish**:

```html
<!-- Inline -->
<button onclick="alert('Salom!')">Click</button>

<!-- Internal -->
<script>
    console.log('Salom!');
</script>

<!-- External -->
<script src="script.js"></script>
```

**Qayerga qo'yish?**:
- `<head>` - sahifa yuklanishidan oldin
- `<body>` oxirida - to'liq yuklanganidan keyin (tavsiya etiladi)

### Ma'lumot turlari

#### Primitive turlar

**1. String (matn)**:

```javascript
let ism = 'Ali';
let familiya = "Valiyev";

```

**Template literal**:
```javascript
let ism = 'Ali';
let yosh = 25;
let matn = `Mening ismim ${ism}, yoshim ${yosh}`;
console.log(matn);  // Mening ismim Ali, yoshim 25
```

**Escape belgilari**:
```javascript
let matn1 = 'Salom\'dan';  // Salom'dan
```

### Operatorlar

#### Arifmetik

```javascript
let a = 10;
let b = 3;

console.log(a + b);  // 13 Qo'shish
console.log(a - b);  // 7  Ayirish
console.log(a * b);  // 30 Ko'paytirish
console.log(a / b);  // 3.333... Bo'lish
console.log(a % b);  // 1  Qoldiq
console.log(a ** b); // 1000 Daraja

a++;  // 11 Increment
b--;  // 2  Decrement
```

**++ va --**:

```javascript
let x = 5;
console.log(x++);  // 5 (keyin oshadi)
console.log(x);    // 6

let y = 5;
console.log(++y);  // 6 (oldin oshadi)
console.log(y);    // 6
```

#### Qiymat berish

```javascript
let x = 10;
x += 5;   // x = x + 5;  (15)
x -= 3;   // x = x - 3;  (12)
x *= 2;   // x = x * 2;  (24)
x /= 4;   // x = x / 4;  (6)
x %= 5;   // x = x % 5;  (1)
x **= 3;  // x = x ** 3; (1)
```

#### Taqqoslash

```javascript
10 == 10;   // true  Teng
10 != 5;    // true  Teng emas
10 > 5;     // true  Katta
10 < 5;     // false Kichik
10 >= 10;   // true  Katta yoki teng
10 <= 5;    // false Kichik yoki teng
```

**== vs ===**:

```javascript
// == (faqat qiymat)
5 == "5";   // true
0 == false; // true
null == undefined; // true

// === (qiymat va tur)
5 === "5";   // false
0 === false; // false
null === undefined; // false
```

**Har doim === ishlatish tavsiya etiladi!**

#### Mantiqiy

```javascript
// && (AND - VA)
true && true;   // true
true && false;  // false
false && false; // false

// || (OR - YOKI)
true || false;  // true
false || false; // false

// ! (NOT - EMAS)
!true;   // false
!false;  // true

// Misollar
let yosh = 25;
let talaba = true;

yosh > 18 && talaba;  // true
yosh < 18 || talaba;  // true
!(yosh < 18);         // true
```

### Sikllar (Loops)

#### for loop

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0 1 2 3 4
}
```

**Qismlar**:
1. Boshlang'ich: `let i = 0`
2. Shart: `i < 5`
3. Qadami: `i++`

```javascript
// Teskari
for (let i = 5; i > 0; i--) {
    console.log(i);  // 5 4 3 2 1
}

// 2 qadam
for (let i = 0; i < 10; i += 2) {
    console.log(i);  // 0 2 4 6 8
}
```

#### while loop

```javascript
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
```

#### do...while loop

```javascript
let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
```

**Farqi**: kamida 1 marta bajariladi

#### break va continue

```javascript
// break - to'xtatish
for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i);  // 0 1 2 3 4
}

// continue - o'tkazib yuborish
for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) continue;
    console.log(i);  // 1 3 5 7 9
}
```
