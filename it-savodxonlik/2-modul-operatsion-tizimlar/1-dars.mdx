## Dars mazmuni

### Windows nima?

**Windows** - Microsoft kompaniyasi tomonidan ishlab chiqilgan dunyodagi eng mashhur operatsion tizim.

**Versiyalari**:
- Windows 7 (eski)
- Windows 8/8.1
- Windows 10
- Windows 11 (eng yangi)

### Windows interfeysi

#### Desktop (Ish stoli)
- Kompyuter yoqilganda ko'rinadigan asosiy ekran
- Fayllar, papkalar, dastur yorliqlari joylashtiriladi
- **Wallpaper (Fon rasmi)** - ish stoli foni

#### Taskbar (Vazifalar paneli)
- Ekranning pastida joylashgan
- **Start tugmasi** - dasturlar menyusi
- **Search (Qidiruv)** - fayl va dastur qidirish
- **Tray (System tray)** - tizim bildirishnomalari, soat
- **Pinned apps** - tez ochish uchun yopishtirilgan dasturlar

#### Start Menu (Boshlash menyusi)
- Barcha dasturlarni ko'rish
- Tez kirish
- Sozlamalar
- O'chirish tugmalari

#### Folders (Papkalar)
- **This PC (Bu kompyuter)** - barcha disklar
- **Documents (Hujjatlar)** - shaxsiy fayllar
- **Downloads (Yuklamalar)** - internetdan yuklab olingan fayllar
- **Pictures (Rasmlar)** - suratlar
- **Videos (Videolar)** - video fayllar
- **Music (Musiqa)** - audio fayllar

### File System (Fayl tizimi)

#### Disklar (Drives)
- **C:** - asosiy disk (Windows o'rnatilgan)
- **D:, E:, F:** - qo'shimcha disklar
- **USB Flash** - tashqi xotira

#### Fayl va papkalar

**Fayl (File)** - ma'lumot saqlangan obyekt
- Nomi va kengaytmasi (extension) bor
- Masalan: `document.docx`, `photo.jpg`

**Papka (Folder)** - fayllarni saqlash uchun konteyner
- Ichida boshqa papkalar bo'lishi mumkin
- Tartibli saqlash uchun ishlatiladi

#### Fayl kengaytmalari (File Extensions)

**Hujjat faylari**:
- `.docx` - Word hujjati
- `.xlsx` - Excel jadvali
- `.pptx` - PowerPoint taqdimoti
- `.pdf` - Portable Document Format
- `.txt` - oddiy matn

**Rasm faylari**:
- `.jpg`, `.jpeg` - siqilgan rasm
- `.png` - shaffof fonli rasm
- `.gif` - animatsiya

**Video fayllar**:
- `.mp4` - video
- `.avi`, `.mkv` - video formatlar

**Audio fayllar**:
- `.mp3` - musiqa
- `.wav` - yuqori sifatli audio

### Asosiy amallar

#### Fayllar bilan ishlash
1. **Yaratish (Create)** - o'ng tugma > New
2. **Ochish (Open)** - ikki marta bosish
3. **Nusxa olish (Copy)** - Ctrl+C
4. **Qirqish (Cut)** - Ctrl+X
5. **Qo'yish (Paste)** - Ctrl+V
6. **Nomini o'zgartirish (Rename)** - F2 yoki o'ng tugma > Rename
7. **O'chirish (Delete)** - Delete tugmasi
8. **Arxivdan qaytarish (Restore)** - Recycle Bin dan

#### Klaviatura yorliqlari (Keyboard Shortcuts)
- `Win` - Start menyusini ochish
- `Win + E` - File Explorer ochish
- `Win + D` - Desktop ni ko'rsatish
- `Win + L` - Kompyuterni qulflash
- `Alt + Tab` - Dasturlar orasida almashinish
- `Alt + F4` - Dasturni yopish
- `Ctrl + Alt + Delete` - Task Manager
- `Win + I` - Settings ochish
