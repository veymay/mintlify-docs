## Dars mazmuni

### Microsoft Excel nima?

**Microsoft Excel** - elektron jadvallar (spreadsheet) bilan ishlash, hisob-kitob qilish va ma'lumotlarni tahlil qilish dasturi.

**Ishlatiladi**:
- Hisob-kitoblar
- Byudjet rejalashtirish
- Ma'lumotlarni saqlash va tartibga solish
- Grafik va diagrammalar
- Statistika va tahlil

### Excel interfeysi

#### Asosiy elementlar:

**Cell (Hujayra)** - ma'lumot kiritiluvchi eng kichik element
- Ustun (Column) va Qator (Row) kesishgan joy
- Masalan: A1, B5, C10

**Row (Qator)** - gorizontal, raqamlar bilan (1, 2, 3...)
**Column (Ustun)** - vertikal, harflar bilan (A, B, C... Z, AA, AB...)

**Worksheet (Ish varag'i)** - bitta varaq
**Workbook (Ish kitobi)** - bir nechta varaqdagi fayl (.xlsx)

#### Ribbon (Lenta) tablari:
- **Home** - formatlash, qo'shish, o'chirish
- **Insert** - jadval, grafik, rasm
- **Page Layout** - sahifa sozlamalari
- **Formulas** - formulalar
- **Data** - ma'lumotlar bilan ishlash
- **Review** - tekshirish va izoh
- **View** - ko'rinish

### Navigatsiya va tanlash

**Harakatlanish**:
- **O'q tugmalari** - hujayra bo'yicha
- **Tab** - o'ngga
- **Enter** - pastga
- **Ctrl + Home** - A1 ga
- **Ctrl + End** - oxirgi hujayra

**Tanlash**:
- **Bitta hujayra** - bosish

- **Hammasi** - Ctrl + A

### Formulalar

**Formula** - hisoblash uchun qoida. Doim `=` belgisi bilan boshlanadi.

#### Asosiy operatorlar:

```excel
+ (qo'shish)        =A1+B1
- (ayirish)         =A1-B1
* (ko'paytirish)    =A1*B1
/ (bo'lish)         =A1/B1
^ (darajaga)        =A1^2
```

#### Hujayra havolasi:

**Relative (Nisbiy)**: `A1`
- Formulani ko'chirganda o'zgaradi

**Absolute (Mutlaq)**: `$A$1`
- Formulani ko'chirganda o'zgarmaydi
- F4 tugmasi bilan

**Mixed (Aralash)**: `$A1` yoki `A$1`
- Faqat ustun yoki qator qotib qoladi

#### Misollar:

```excel
=A1+B1+C1          (Uchta sonni qo'shish)
=A1*B1             (Ikki sonni ko'paytirish)
=(A1+B1)/2         (O'rtacha qiymat)
=A1*10/100         (10% ni hisoblash)
```

### Funksiyalar

**Funksiya** - tayyor formula.

#### Asosiy funksiyalar:

**1. SUM (Yig'indi)**
```excel
=SUM(A1:A10)       (A1 dan A10 gacha yig'indisi)
=SUM(A1:A5,C1:C5)  (Ikki diapazon yig'indisi)
```

**2. AVERAGE (O'rtacha)**
```excel
=AVERAGE(A1:A10)   (O'rtacha qiymat)
```

**3. MAX va MIN (Eng katta va kichik)**
```excel
=MAX(A1:A10)       (Eng katta qiymat)
=MIN(A1:A10)       (Eng kichik qiymat)
```

**4. COUNT (Sanash)**
```excel
=COUNT(A1:A10)     (Raqamlar sonini sanash)
=COUNTA(A1:A10)    (Bo'sh bo'lmagan hujayralar)
```

**5. IF (Agar)**
```excel
=IF(A1>50,"O'tdi","Yiqildi")
=IF(A1>=90,"A'lo",IF(A1>=70,"Yaxshi","Qoniqarsiz"))
```

**6. VLOOKUP (Vertikal qidirish)**
```excel
=VLOOKUP(A1, B1:D10, 2, FALSE)
```

**7. Sana funksiyalari**
```excel
=TODAY()           (Bugungi sana)
=NOW()             (Hozirgi vaqt)
=YEAR(A1)          (Yil)
=MONTH(A1)         (Oy)
=DAY(A1)           (Kun)
```
